### TS2307 false positive em `topico-to-categoria.ts`

Diagnóstico paralelo executado durante o E2E confirmou:

| Verificação | Resultado |
|---|---|
| `server/config/topico-to-categoria.ts` existe | ✅ |
| 6 importadores válidos | ✅ (2 analyzers + 3 tests + auto) |
| `npx tsc --noEmit` | ✅ EXIT=0 |
| Histórico git | ✅ criado em `5765a93`, squash `4dccbbc` (PR #976 Fix B) |

**Causa raiz:** race condition do `git reset --hard` (working tree atualizado arquivo-a-arquivo, watchers detectam estado intermediário). Log do dev server confirmou: `[tsx] unlink in ./server/config/topico-to-categoria.ts Restarting...`. Sem ação no código.

**Sugestão de prevenção (backlog opcional):** health check usar `tsc --noEmit` one-shot ao invés de output streaming de `tsc --watch`, ou debounce explícito (ignorar erros TS em janela < N segundos desde último filesystem event).
